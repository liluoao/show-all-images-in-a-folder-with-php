### 设置

```php
# 图片文件夹路径
$imageFolder = 'img/';

# 需要展示的类型
$imageTypes = '{*.jpg,*.JPG,*.jpeg,*.JPEG,*.png,*.PNG,*.gif,*.GIF}';

# 是否按名称排序
# 否则按时间排序
$sortByImageName = false;
```
